ClassifierType¶

class Classification.ClassifierType[source]¶

Bases: enum.Enum

Defines accepted classifier type as required in this application

Attributes

BinaryGenres (int) Genre is either Fantasy or Non-Fantasy
MultiClass (int) Genre can be multiple classes from dark fantasy to juvenile fantasy
NoClass (int) Default genre this should not result in any processing
